# 链表

## 链表反转

```java
// 迭代
public ListNode reverse(ListNode head) {
    ListNode prev, curr, next;
    prev = null;
    curr = next = head;
    while (curr != null) {
        next = curr.next;
        curr.next = prev;
        prev = curr;
        curr = next;
    }
    return prev;
}

// 递归
public ListNode reverse(ListNode head) {
    if (head == null || head.next == null) return head;
    ListNode last = reverse(head.next);
    head.next.next = head;
    head.next = null;
    return last; 
}
```
